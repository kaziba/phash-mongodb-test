primer-phash-mongodb
======

# Usage

    git clone https://github.com/kaziba/phash-mongodb-test.git
    cd phash-mongodb-test
    npm run setup
    npm run start

# Requirement

- pHash
- GraphicsMagick

# Install

On Debian:

**pHash**

    sudo apt-get install cimg-dev libphash0-dev libmagickcore-dev

See below for details
